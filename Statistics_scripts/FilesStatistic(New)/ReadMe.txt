FilesStatistic(WithDuplicateInfo).vbs - обновлённый скрипт, который дублирует отправляемую информацию в виде набора кодов символов.

Для расшифровки искажённой информации следует скопировать цифровую часть сообщения и сохранить её в файле "Mail.txt", после чего запустить скрипт DecoderMail.vbs.
В результате получится файл Result.txt, в котором будет сообщение в расшифрованном виде.
