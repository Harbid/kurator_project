1). TPB_parser.vbs - скрипт для сбора статистики размеров раздач с указанного раздела на The Pirate Bay

Входным параметром является файл с набором ссылок (resources.txt) на разделы TPB. Каждый раздел имеет 100 страниц информации, которая показывает последние раздачи за 48 часов.

В результате работы скрипта будет сгенерировано два файла:
 - Log.txt    : в нём можно увидеть, какую страницу обрабатывает скрипт в текущий момент
 - Result.txt : в этот файл выводится статистика по обработанным страницам
