1). URLCollector.vbs - вспомогательный скрипт для сбора ссылок на раздачи d2.vu для последующего их сканирования. 

В результате работы скрипта будет сгенерирован файл:
 - d2.vu_torrent_ulrs.txt : в нём содержится список ссылок на раздачи.

(опытным путём выяснено, что раздач на сайте всего только 10000)

2). d2.vu_parser.vbs  - скрипт для сбора статистики размеров файлов из раздач на d2.vu.

Входным параметром для работы скрипта является файл "d2.vu_torrent_ulrs.txt".
В результате работы скрипта генерируются два файла:
 - Log.txt    : в нём можно увидеть, какую страницу обрабатывает скрипт в текущий момент . Записывается время обработки, полученная ссылка на страницу со списком файлов и промежуточные результаты статистики размеров файлов.
 - Result.txt : в этот файл выводится итоговая статистика размеров файлов с раздач d2.vu
