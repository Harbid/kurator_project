1).TorrentParser(Single).vbs - скрипт для сбора статистики размеров файла с указанной ветки форума на rutracker.org

Входным параметром ("StartPage") является начальная страница форума.
В результате работы скрипта будет сгенерировано два файла:
 - Log.txt    : в нём можно увидеть, какую страницу обрабатывает скрипт в текущий момент
 - Result.txt : в этот файл выводится статистика по обработанным страницам

2). ForumsChecker.vbs - скрипт для формирования статистики форумов rutracker.org
Входных параметров нет.
В результате работы скрипта будет сгенерировано два файла:
 - Valid_Forums.txt : В этом файле формируется список ссылок на форумы, в которых присутствует требуемая информация о торрентах
 - Error_Log.txt    : Файл для ведения лога ошибок при проверке форумов. Всего есть 3 вида ошибок: 
 --- Error: Такого форума не существует.
 --- Error: Требуется логин/пароль.
 --- Error: На форуме отсутствуют торрены.

Проверка форумов ведётся до тех пор, пока не будет превышен установленный предел повтора ошибок (т.е. больше нет действующих форумов).
